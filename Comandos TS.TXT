____________________________________________________________________
(Comandos Type Script):

Instalar o typescript de forma global:

npm i -g typescript
____________________________________________________________________
Checar Versão typescript:

tsc -v
____________________________________________________________________
Criar Arquivo de Configuração typescript:  

tsc --init
____________________________________________________________________
Compilação Manual:

tsc
____________________________________________________________________
Compilação automatica para cada atualização no arquivo TS: 
Aqui tudo que for mudado no arquivo ts já será compilado para o JS:

tsc -w
